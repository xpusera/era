Obelisk Analog Features  

Entity Behaviors  
1. Stalking - Follows player from a distance, stays hidden  
2. Chasing - Pursues player quickly when triggered  
3. Window Peeking - Watches player from windows  
4. Corner Watching - Observes player from dark corners  
5. Ceiling Hanging - Hangs upside down from ceilings  
6. Floor Emerging - Rises from the ground with dirt particles  
7. Wall Phasing - Appears through walls in player’s view  
8. No Escape Mode - Relentless pursuit, hard to escape  
9. Crazy Mode - Erratic movement with glowing effects  

Entity Abilities  
10. Whispering - Sends creepy red chat messages  
11. Teleportation - Moves instantly closer to player  
12. Invisibility - Becomes invisible or flickers in/out  
13. Speed Manipulation - Alters movement unpredictably  
14. Torch Destruction - Breaks nearby torches to darken area  
15. Door Interaction - Opens/closes doors, appears behind them  
16. Sound Distortion - Plays random eerie sounds  
17. Item Stealing - Removes items from inventory  
18. Inventory Shuffling - Randomizes inventory slots  
19. Behind Spawning - Teleports directly behind player  
20. Light Flickering - Torches flicker on and off  
21. Player Freezing - Immobilizes player temporarily  
22. Fog Generation - Creates dense fog around player  
23. Jumpscares - Fullscreen scary image with sound  
24. Environment Manipulation - Alters time, destroys leaves  
25. Size Changing - Randomly grows or shrinks  
26. Clone Surrounding - Spawns fake copies around player  
27. Mirror Appearing - Appears as ghostly white version  
28. Block Decay - Destroys nearby blocks  
29. Name Whispering - Whispers player’s username  
30. Block Breaking - Breaks blocks in crazy mode  

Phases System  
31. Phase 1 Awakening - Spawns at night, low activity  
32. Phase 2 Emergence - Spawns during day, stronger abilities  
33. Phase 3 Pursuit - Aggressive, higher kill chance  
34. Phase 4 Terror - Darkness particles, frequent sounds  
35. Phase 5 Nightmare - Maximum danger, torch flickering  

Screen Effects  
36. Screen Tearing - Horizontal colored lines  
37. Desaturation - Gray overlay reduces color  
38. Vignette - Dark screen borders  
39. Screen Flash - Sudden bright flashes  
40. Heartbeat Effect - Rhythmic tension sound  
41. Vision Distortion - Warped field of view  
42. View Shaking - Violent camera shake  

Special Events  
43. Blood Moon - Midnight, max phase, red flash  
44. The Watching - Multiple clones spawn  
45. Time Slip - Random time changes  
46. Sudden Appearance - Appears directly in front  

Dimensions  
47. The Sequence - Brick grid dimension  
48. Wonderland - Foggy forest dimension  
49. The Corners - Dark cobblestone maze  

Dimension Features  
50. God Mode Protection - Temporary immortality  
51. Void Respawn - Teleports back from void  
52. Time Control - Dimensions alter time  
53. Dimension Spawning - More frequent entity spawns  

Portals  
54. Portal Activator - Craftable travel block  
55. Portal Keys - Keys for each dimension  
56. Dimension Portals - Colored glowing travel blocks  
57. Portal Particles - Floating portal particles  

Structures  
58. Abandoned Houses - Decayed buildings with loot  
59. Creepy Shrines - Ritual stone platforms  
60. Warning Signs - Signs with danger text  
61. Ritual Circles - Stone circles with activator  
62. Grave Sites - Dirt mounds with markers  
63. Ruined Towers - Tall decayed towers with loot  
64. Creepy Wells - Dark stone wells  

Custom Nodes  
65. Ritual Stone - Glowing obsidian shrine block  
66. Warning Sign - Wooden danger sign  
67. Shrine Block - Stone brick for shrines  
68. Dead Leaves - Dried leaves from abilities  

Ambient Features  
69. Night Empowerment - Faster at night  
70. Ambient Sounds - Random creepy noises  
71. Paranoia System - Tracks fear and stolen items  
72. Persistent Data - Saves phase and structures  

Admin Commands  
73. horror_spawn - Force entity spawn  
74. horror_phase - Set phase 1–5  
75. horror_debug - Show debug info  
76. horrortp[dimension] - Teleport to dimension  
77. horrorspawn[structure] - Spawn structure at location  
78. horrortest[effect] - Test screen effects
